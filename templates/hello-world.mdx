The Hello World template is the simplest Electrobun app. It demonstrates the basic structure and core features you need to get started.

## Features

- Basic window creation
- Simple HTML/CSS UI
- Views protocol for loading local files
- Window event handling

## Initialize the Template

Clone the template from the repository:

```bash
git clone https://github.com/blackboardsh/electrobun.git
cp -r electrobun/templates/hello-world my-app
cd my-app
bun install
```

Run the app:

```bash
bun run dev
```

## Key Code

### Creating a Window

The main process creates a window and loads the UI:

```typescript src/bun/index.ts
import { BrowserWindow, Utils } from "electrobun/bun";

// Create the main application window
const mainWindow = new BrowserWindow({
  title: "Hello Electrobun!",
  url: "views://mainview/index.html",
  frame: {
    width: 800,
    height: 800,
    x: 200,
    y: 200,
  },
});

// Quit the app when the main window is closed
mainWindow.on("close", () => {
  Utils.quit();
});
```

### Configuration

The `electrobun.config.ts` file defines the app structure:

```typescript electrobun.config.ts
import type { ElectrobunConfig } from "electrobun";

export default {
  app: {
    name: "hello-world",
    identifier: "helloworld.electrobun.dev",
    version: "0.0.1",
  },
  build: {
    views: {
      mainview: {
        entrypoint: "src/mainview/index.ts",
      },
    },
    copy: {
      "src/mainview/index.html": "views/mainview/index.html",
      "src/mainview/index.css": "views/mainview/index.css",
    },
  },
} satisfies ElectrobunConfig;
```

### Loading Local Files

The `views://` protocol loads files from your app bundle:

```typescript
url: "views://mainview/index.html"
```

This maps to the file copied in the build configuration:

```typescript
copy: {
  "src/mainview/index.html": "views/mainview/index.html",
}
```
